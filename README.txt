ASTROLABIO RECASTER
Versão: 1.0
Autor: Nathan Corrêa (iTzCROW)

DESCRIÇÃO
Astrolabio Recaster é uma ferramenta para leitura automática dos atributos do sistema de Astrolábio no jogo Perfect World, com OCR, comparação de atributos desejados e automação de roletagem.

REQUISITOS
- Windows 64-bit
- O jogo deve estar aberto em modo janela ou modo compatível com captura de tela
- A pasta "tessdata" deve estar no mesmo local do executável
- O arquivo "eng.traineddata" deve estar dentro da pasta "tessdata"

ESTRUTURA DA PASTA
Mantenha estes arquivos juntos:
- Astrolabio Recaster.exe
- pasta tessdata
- README.txt
- demais arquivos gerados no publish, se existirem

COMO USAR
1. Extraia todos os arquivos para uma pasta de sua preferência.
2. Execute "Astrolabio Recaster.exe".
3. Abra o Astrolábio no jogo.
4. Posicione a janela "Região de captura" do programa sobre a área dos atributos e clique em "Teste de Captura", o programa deve ler corretamente os 10 atributos (ou quantos houverem no astrolabio) antes de prosseguir. Caso a leitura não seja feita corretamente, reposicione a área de captura até que o programa leia corretamente.
5. Clique em "Calibrar Mouse" e, quando o programa sumir, posicione o cursor exatamente sobre o botão "Iniciar Horóscopo" do jogo.
6. Selecione os atributos desejados e as quantidades.
7. Defina o número máximo de tentativas ou marque "Sem limite".
8. Clique em "Iniciar" para começar a automação.

PARAR A AUTOMAÇÃO
- Use o botão "Parar"
- O programa utiliza a macro do mouse para clicar no botão dentro do jogo, então, caso seja necessário interromper a automação e não seja possível clicar no botão PARAR, utilize a tecla F8 para finalizar a automação.

OBSERVAÇÕES IMPORTANTES
- O programa depende do posicionamento correto da janela sobre a área dos atributos.
- Mudanças na resolução, escala da interface do jogo ou posição da janela podem exigir nova calibração.
- Se o OCR falhar, faça novo teste de captura e reposicione o programa.

SOLUÇÃO DE PROBLEMAS
- Se o programa não abrir: verifique se todos os arquivos do publish foram extraídos.
- Se o OCR não funcionar: confira se "tessdata/eng.traineddata" está presente.
- Se o clique não funcionar: refaça a calibração do mouse.
- Se a leitura falhar: reposicione a janela do programa exatamente sobre a área dos atributos.
